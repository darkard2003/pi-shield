# Pi Shield Extension Project

This project is a security guardrail extension for the `pi-coding-agent`. It implements path traversal protection, shell command restrictions, and custom secure tools (ls, find, grep) to prevent dangerous actions.

## Documentation
For more information on how extensions work in `pi-coding-agent`, see the documentation at:
- `~/.nvm/versions/node/v22.19.0/lib/node_modules/@earendil-works/pi-coding-agent`
